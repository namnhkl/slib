# Env
## Required
- Nodejs 22
- Npm 11 (yarn 3.6)

## Start
```bash
npm install (yarn install)
```

## Run Dev
```bash
npm run start (yarn start)
```

## Run Build
```bash
npm run build (yarn build)
```

## Sync Multi Language
```bash
npm run s (yarn s)
```

## make page

```bash
npx ng generate component pages/<page>
```

## make module

```bash
npx ng generate module <name> --route
```
